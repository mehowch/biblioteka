Technologies used: Spring Boot, Spring Boot Security, Swagger2, Thymeleaf, Maven, MySQL DB, H2 DB(Heroku).
Documentation: Spring Security provides safe login/logout process as well as application layer functionality access. All user passwords are encrypted and safely stored inside database.
Application provides two user roles:
  • Admin - creates new users, manages book storage (creates new books, deletes old books, updates books)
  • User - borrows books, returns books
Presentation layer created with help of Thymeleaf allows user to access user gui and easily search for books to borrow. User may access User panel and see all currently borrowed books. Endpoints:
  • Swagger UI Documentation
  • H2 Console
